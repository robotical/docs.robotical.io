Integration and API Docs for Marty V2


  Marty V1 Docs


Marty V2 can be controlled using your smart devices bluetooth connection. Marty's controller board (RIC) supports Bluetooth Version 4.2 and newer. Support for Raspberry Pi, Arduino and other microcontrollers will be added in the coming months. Support for Wifi and USB will be added

Some of our API's and integration will only work in some of these modes, so we have split them into groups bellow.

Scratch
Beginner

Scratch is the simplest way to program a Marty V2 if you're a complete beginner. It has a simple graphical interface and blocks you can combine to build up a program, and is very popular with kids & educators.


Hardware & Low-Level APIs


This set of articles cover the sensors and physical aspects of Marty, as well as the more nitty-gritty technical details of RIC, Marty's control board and the associated electronics they come with. Also covered are complete connection details for interfacing with Robotical Add-ons.

Additionally our lowest level APIs are documented here, so if there's a language or system you want to use that's not currently supported, you may still be able to implement your control through these bare-metal APIs.
